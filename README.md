***Зависимости "pip install requirements.txt"***

>У групп должно быть минимут 100 подписчиков
>
>И создана должна быть не менее 2 недель назад

```
config = dict(tokens=['первый токен',
                      'второй токен',
                      'третий',
                      'и так далее',
                      'хоть 100'
                      ])
```
